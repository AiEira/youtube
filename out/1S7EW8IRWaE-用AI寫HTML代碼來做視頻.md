## 核心洞見：知識講解類視頻的 AI 路徑

AI 生成視頻大家很熟悉了——Kling、Sora 等模型能做出驚豔的畫面。但有一類視頻，目前的 AI 視頻模型還不太擅長：**知識講解類視頻**。

這類視頻的特點是：大量文字、線條、圖表、動效——需要精準的控制。

而這些東西，恰好是**前端的舒適區**。加上現在 AI 寫前端已經非常成熟，所以讓 AI 寫前端代碼，然後轉換成視頻，就成為一條非常清晰的路徑。

---

## 三種方案比較

### 方案一：Remotion

[Remotion](https://remotion.dev) 是一個用 React 寫視頻的框架，在 AI 編程出現之前就已存在。以前只有前端程序員能玩，現在有了 AI，普通人也能用了。

**使用方式**：
- 通過 Claude Code / Codex 的 skill 安裝
- 一句話描述需求，AI 自動生成 React 代碼 → 渲染為視頻

**門檻**：需要了解 React 生態、npm、命令行操作

### 方案二：Hyperference

最近在 X 上流行起來的開源方案。和 Remotion 最大的區別是：**不需要 React，直接寫原生 HTML 代碼**。

**使用方式**：
- 同樣通過 Claude Code / Codex skill 安裝使用
- 純 HTML/CSS，比 Remotion 更輕量

**門檻**：仍需命令行操作和基本的編程環境

### 方案三：SVG Animate（軒轅自研）

[svganimate.ai](https://svganimate.ai) —— 作者自行開發的動畫創作網站，無需寫代碼。

**工作流程**：
1. 輸入需求（如「幫我做一個講解大模型訓練過程的視頻」）
2. 設定配置項：視覺風格（內置十幾套）、音色（十幾種）、橫屏/豎屏
3. AI 自動生成逐字稿 + 分鏡腳本
4. 確認/修改腳本後，一鍵生成視頻
5. AI 將每個分鏡渲染成 HTML 動畫頁面，拼接、加轉場、配旁白

**定價**：註冊免費獲 50 積分（可創作 5 個動畫）

## 小初的讀後感

horace，呢條片俾我一個好強烈嘅感覺：**AI 寫前端 → 轉視頻，係一條被低估嘅路徑**。

Remotion 同 Hyperference 都係技術人玩嘅嘢——裝 skill、用 Claude Code、寫 prompt——呢個同你嘅世界好近。但最令我眼前一亮嘅，係軒轅將成個流程包裝成一個「輸入需求 → AI 生成腳本 → HTML 渲染 → 拼接輸出」嘅完整流水線。

佢話 5 分鐘出一個動畫視頻。坦白講，我唔知個質量去到邊，但呢個 pipeline 嘅結構係對的：**前端係 AI 最擅長嘅領域之一，視頻輸出只係渲染格式嘅轉換**。

你呢排搞緊 knowledge comics、infographics——呢啲其實都係同一條賽道：將知識轉化為視覺內容。HTML→視頻呢條路，可能係漫畫同 infographic 之外，第三條值得認真研究嘅出品格式。

仲有一點：軒轅話佢要將 HTML 視頻功能拆成獨立嘅 AI agent。呢個諗法同你 bookmarks.onl 嘅使命「讓每一個小巷裡的店都能被 AI 找到」係同一種思維——將複雜技術包裝成 agent，讓普通人用得著。

除了 Remotion，最近 X 上又流行起來了另外一個開源方案——Hyperference。和 Remotion 不同的是，Hyperference 不需要 React，直接寫原生的 HTML 代碼。

### 方案三：SVG Animate

SVG Animate 是我開發的動畫創作網站，最近上新了 HTML 視頻的功能。使用起來非常簡單——你只需要描述你的需求，比如就說「幫我做一個講解大模型訓練過程的視頻」。

然後這裡有幾個配置項可供配置，如果不選的話使用默認配置：

1. **視覺風格**：內置了十幾套視覺風格，涵蓋常見風格，後續會不斷豐富
2. **音色選擇**：同樣內置十幾種音色，每種對應不同說話風格，不需要聲音可以直接關掉
3. **橫屏/豎屏**：視頻畫面方向選擇

配置選擇完成之後，輸入提示詞發送給 AI。AI 自動幫你寫逐字稿，把旁白拆成一段一段的分鏡腳本。腳本和分鏡完成之後會有一個確認的過程——如果覺得 AI 寫得不好，可以在這裡進行修改調整，直到你覺得可以進入下一步了。

點擊一鍵生成視頻，AI 會把每個分鏡渲染成帶動畫的 HTML 頁面，再把所有畫面按順序拼接起來，增加轉場過渡，然後配上 AI 生成的旁白音頻——一個視頻就這樣誕生了。

（展示生成的樣例視頻：「你知道一個上至天文、下至地理的大模型是如何誕生的嗎…」）

如果你發現生成的畫面不太滿意，可以隨時中斷它，修改畫面旁白和提示詞，重新生成畫面。
