## 一句總結

AI 用寫代碼的方式做視頻，才是 AI 時代的剪輯方式。寫一次，永久復用——而且 AI 最擅長的就是寫代碼。

---

## 6 個實戰 Demo 速覽

### Demo 1：截圖復刻動畫
截一張「小 Lin 說」土耳其經濟動畫的圖，發給 Claude Code，它自己分析動畫邏輯，用 **Remotion**（前端動畫庫）寫代碼做出來。國旗飄揚、虛線折線、小船卡車來回跑、總統人物搖擺——全部代碼生成。

### Demo 2：錄屏 → 完整動畫 + 配樂
錄 2 秒動畫片段發給 Claude Code → 分析每個元素的動畫邏輯 → 代碼復刻。配樂用 **Epidemic Sound** 的 AI 匹配功能找風格相近的音樂，再讓 Claude Code 加進去。

### Demo 3：扔影片連結直接復刻
發一個 YouTube 影片連結給 Claude Code，指定「復刻第 6–18 秒的動畫」，Claude 自己下載觀看、解析、寫代碼做出來。

### Demo 4：章節進度條 Skill
把字幕檔發給 Claude Code，自動生成影片章節進度條動畫。做了一次後封裝成 **Claude Skill**，之後一句話就能調用。

### Demo 5：音效 + 字幕自動化
把口播常用音效和對應的文字動畫效果整理好，Claude Code 自動識別哪些地方需要什麼效果並匹配；甚至可以自動加「抖音風格字幕」。

### Demo 6：打字機板書 B-roll
把字幕和素材發給 Claude Code，自動判斷哪裡需要劃重點、加什麼素材，生成一個字一個字蹦出來 + 機械鍵盤敲擊聲的打字機板書效果（來自 YouTuber Ray 的 `typewriter-video` Skill）。

## 核心邏輯

1. **影片的本質**：每秒 24–60 張圖快速翻動
2. **網頁 = 代碼畫的圖**，上千張網頁疊在一起就是動畫
3. **傳統剪輯**：每次拖拽，做完就沒了；**代碼方式**：寫一次，改參數就能永久復用
4. AI 最擅長寫代碼 → AI + 代碼 = 結果導向的剪輯

## 關鍵心態轉變

> 「這不是舊工作流的升級，而是底層邏輯徹底的重寫。」

心心用 iPhone 去掉實體鍵盤來比喻——當時所有人都說不可能，但去掉鍵盤反而解鎖了更大的屏幕和更多可能性。用寫代碼和說話的方式做影片，現在可能還磕磕絆絆，但方向已經很清晰。

> 「做影片最重要的從來都不是剪輯，而是你想表達什麼。剪輯只是一個輔助手段。」

把大量時間花在找素材、做動畫、加音效、做字幕上，消耗的是你本該用來思考和表達的時間。如果能標準化、做成 Skill，讓剪輯越來越輕，省下來的精力就能回歸創作本身。
