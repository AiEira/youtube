## 是甚麼

Anthropic 的 Claude Design 能一鍵生成高品質設計，但有兩個致命限制：必須是 Claude 付費用戶，而且 token 消耗極大——做一個小任務就可能耗盡額度。

影片作者把 Claude Design 的工作機制逆向工程，做成了開源 skill **「Huashu Design」（畫書設計）**，GitHub 上已獲得 16,000+ stars。它可以在任何 coding agent 中使用：Claude Code、Cursor、Codex、Hermes、Trae、QClaw、Workbuddy——幾乎所有支援 skill 的工具都能裝。

---

## 核心能力

| 能力 | 說明 |
|------|------|
| **PPT 生成** | 網頁展示 + PPTX + PDF 三種輸出，格式維持，內容可編輯 |
| **網頁設計** | 個人主頁、Landing page、產品頁——完整動畫與互動 |
| **互動原型** | iOS/Android APP 可操作原型，快速驗證產品想法 |
| **宣傳動畫** | 15-20 秒產品宣傳片，自動抓取品牌色與 LOGO |
| **圖片驅動設計** | 給一張照片/海報/藝術作品 → 自動抽取色系 → 生成 brand spec → 依風格設計 |

---

## 工作流程

Huashu Design 不是「你一句話，它直接產出」。它有一套結構化流程：

1. **需求確認**：先問聽眾背景、目標、視覺風格偏好
2. **風格選擇**：提供多個視覺方案供選擇（也可投餵圖片自訂風格）
3. **資訊收集**：需要時自動搜尋相關知識（如產品介紹、品牌色）
4. **先做兩頁預覽**：讓用戶確認風格後再繼續
5. **自動檢查**：生成後寫程式截圖驗證設計，發現問題自動修復
6. **多格式輸出**：PPTX、PDF、HTML 全部可選

## 四個 Demo

### Demo 1：PPT——向非技術人員講解 Coding Agent

任務：30 分鐘演講，向白領工作者介紹「什麼是 coding agent / Claude Code」。

流程：選定聽眾背景 → 選擇視覺風格 → agent 先做兩頁預覽確認 → 繼續生成 → 自動檢查修復 → 輸出網頁版 + PDF + PPTX。

結果：80 分水準，Claude 風格配色，PPTX 格式維持完美，內容可編輯。

### Demo 2：個人主頁——用一張飲料店照片驅動設計

任務：製作個人主頁，用於社交場合分享。

關鍵操作：投餵一張在飲料店前拍的照片 → agent 抽取照片中的色系（藍色 + 抹茶綠）→ 生成詳細 brand spec（色值、字形、禁區、核心元素）→ 基於 brand spec 設計主頁 → 截圖驗證 → 提建議（換真人照片替代印章）。

結果：視覺高度個人化，色系忠實反映投餵圖片。

### Demo 3：iOS APP 互動原型——《高效能人士的 7 個習慣》

任務：基於書中理念，製作一個管理個人 to-do 的 iOS APP 原型。

流程：選擇視覺氣質 → 選擇功能深度 → agent 自行理解書中內容 → 設計四象限矩陣（影響圈/關注圈）→ 可互動操作。

洞察：這個能力對產品驗證極其重要——「你可能瞬間發現這個東西沒有做的價值，或者發現不錯但想做得不太一樣」。

### Demo 4：宣傳動畫——B 站「花生 AI」

任務：為 B 站新推出的 AI 影片產品「花生」製作 15-20 秒宣傳動畫，16:9 格式。

流程：agent 先確認產品存在 → 自動抓取官網 LOGO 與品牌色 → 生成品牌 DNA 文檔 → 確認動畫內容方向（全流程一鍵成片）→ 製作動畫。

結果：85 分，完全使用花生品牌色與 LOGO，動畫流暢。作者特別提到：這個能力也可用於為自己錄製的影片製作 B-roll 素材。

## 小初的讀後感

這個 skill 的價值不在於「又一個設計工具」，而在於它解決了一個關鍵痛點：**Vibe coding 產出的設計太爛**。AI 寫程式很強，但大多數 AI 生成的網站/APP 都在用同一套預設色系，千篇一律。

Huashu Design 真正聰明的地方是那個**圖片驅動的品牌抽取**——你給它一張照片，它不是簡單取幾個顏色，而是生成完整的 brand spec（色系、字形、禁區、核心數據），然後嚴格依照 spec 設計。這等於把品牌設計師的思考流程編碼進了 skill。

對 horace 來說，這個 skill 有直接實用價值。如果將來要為 AI 小巷計劃的任何項目做 Landing page、PPT 演示、產品動畫——不需要從零 prompt，直接用 Huashu Design 就能產出有水準的設計。

另外，作者提到這個 skill 支援 **Hermes**，這是第一次有外部開發者明確將 Hermes 列為相容平台。值得留意 Huashu Design 在 Hermes 上的實際表現。

现在的话我们可以看到它的这个页面已经做好了。页面做好之后的话，它其实还有一个检验的过程——它会去调用去写代码去进行整个网页的截图，去验证它的设计到底有没有问题，然后如果有问题的话它会再去做一些修改。

他已经在帮我基本完成和实现这个动画了。我觉得这个动画还挺有意思的——它真的用的完全是花生品牌的 LOGO。你看这个是它整体的过程：「请喂我花生」。我觉得他甚至达到了 85 分的我的期待水平。你能用一句话就能产生这么一段去宣传你产品的动画，我觉得这真是一个非常惊喜的东西。

所以如果你现在平时在做一些 Vibe Coding 的事情，你有很多你自己的产品想要去宣传的话，我真的很建议你用我的 Huashu Design 去制作一个宣传的东西，在你的社交媒体上进行宣传。而且除了做宣传动画之外，他也完全可以——就是你录完一段 B 站视频之后，如果你没有那种 B-roll 的素材的话，其实你也完全可以让他去帮你根据你讲解的内容去制作一些动画。
