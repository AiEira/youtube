## 一、這是什麼？

**MoneyPrinterTurbo**（開發者：harry0703）是一個開源 AI 影片生成工具。輸入一個關鍵字，自動產出一支帶旁白、字幕、配樂的高清短影音。

## 二、傳統短影音製作的痛點

| 步驟 | 傳統方式 | MoneyPrinterTurbo |
|------|---------|:--:|
| 文案構思 | 手動撰寫 | LLM 自動生成 |
| 配音 | 錄製或外包 | TTS 語音合成 |
| 素材 | 手動搜尋/購買 | Pexels API 自動抓取 |
| 剪輯 | 手動剪輯軟體 | moviepy 自動合成 |
| 字幕 | 手動上字幕 | edge-tts / whisper 自動 |

---

## 三、技術架構

採用 **MVC 架構**（模型-視圖-控制器分離），自動化管線：

```
關鍵字 → LLM 生成文案 → TTS 語音合成（記錄時長）
     → Pexels API 搜尋無版權素材
     → moviepy 合成（裁剪、疊字幕、配樂）
     → 輸出完整影片
```

**兩種字幕方案**：
- `edge-tts`：快速、輕量
- `whisper`：高品質、較耗資源
