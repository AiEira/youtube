## 一句總結

DeepSeek V4 完成從 NVIDIA CUDA 到華為昇騰（Ascend）的全站遷移——200+ 核心算子重寫、推理成本降至 NVIDIA 方案 1/4、API 定價僅 GPT-5 的 1/500–1/700，為國產 AI 生態打下完整閉環。

---

## 第一部分：遷移難度——CUDA → 昇騰

### 關鍵挑戰

| 挑戰 | 說明 |
|------|------|
| **算子不相容** | CUDA 算子無法直接在昇騰上跑 |
| **Flash Attention 等新算子** | Model-specific 算子需要手動重寫 |
| **精度對齊** | 不同硬體浮點運算順序不同，誤差在長上下文推理中被放大 |
| **逐層比對** | 只能用工具逐層比對、逐層調整 |

> 「這個是非常耗時間的，而且是非常容易出錯的一個過程。」

---

## 第二部分：全站協同創新

### DeepSeek V4 做了什麼

- **200+ 核心算子全部重寫**：融合 kernel + 多流並行，計算與通信極致優化
- **KV Cache 昇騰專屬池化管理**：延遲降、吞吐量升
- **GRPO 強化學習算法**：訓練成本直接減少 40%
- **華為超節點架構**：高效分佈式推理
- **FP4 極致量化**：模型大小縮 4 倍，精度幾乎無損

### 性能數據

| 指標 | 數值 |
|------|------|
| V4 Pro 單卡解碼 | **4,700 tokens/sec**（NVIDIA H200 的 2 倍+） |
| 輕量版 | **6,000+ tokens/sec** |
| 推理成本 | NVIDIA 方案的 **1/4** |
| API 定價 | GPT-5 的 **1/500 – 1/700** |

## 第三部分：國運博弈——戰略意義
